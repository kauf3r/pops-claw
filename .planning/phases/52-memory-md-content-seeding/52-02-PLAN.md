# Plan 52-02: Improve Flush Prompt

## Objective
Replace generic flush prompt with structured template including SQL queries for DB state, producing 10+ line daily summaries.

## Tasks

<task id="1" name="Design structured flush prompt">
Create prompt with sections:
- ## Session Summary (what happened)
- ## DB State (concrete counts from each DB)
- ## Decisions Made (any decisions from this session)
- ## Open Items (unresolved tasks or questions)
Include specific sqlite3 queries for all 6 DBs:
- health.db: health_snapshots count
- observability.db: llm_calls count today

Quiet day minimum: 5 lines with DB snapshot
Active day target: 10+ lines
</task>
