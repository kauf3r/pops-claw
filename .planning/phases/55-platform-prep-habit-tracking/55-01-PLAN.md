2. **Add growth.db bind-mount and OPENCLAW_TZ using Python (safe JSON manipulation):**
   ```python
   python3 -c "
   import json
   path = '/home/ubuntu/.openclaw/openclaw.json'
   with open(path) as f:
       cfg = json.load(f)

   # Add growth.db bind-mount
   binds = cfg['agents']['defaults']['sandbox']['docker']['binds']
   new_bind = '/home/ubuntu/clawd/growth.db:/workspace/growth.db'
   if new_bind not in binds:
       binds.append(new_bind)
       print(f'Added bind-mount: {new_bind}')
   else:
       print('Bind-mount already exists')

   # Add OPENCLAW_TZ to Docker env
   docker_cfg = cfg['agents']['defaults']['sandbox']['docker']
   if 'env' not in docker_cfg:
       docker_cfg['env'] = {}
   docker_cfg['env']['OPENCLAW_TZ'] = 'America/Los_Angeles'
   print('Set OPENCLAW_TZ=America/Los_Angeles')

   with open(path, 'w') as f:
       json.dump(cfg, f, indent=2)
   print('Config saved.')
   "
   ```

3. **Verify config changes:**
