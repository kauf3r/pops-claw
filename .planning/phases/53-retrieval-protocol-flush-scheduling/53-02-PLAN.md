# Plan 53-02: Reschedule Daily Flush to End-of-Day

## Objective
Add a dedicated daily flush cron job at 23:00 UTC (4pm PT) so daily summaries capture the full day's activity.

## Tasks

<task id="1" name="Check current cron configuration">
SSH to EC2 and check how crons are configured in openclaw.json.
Look at agents.defaults.compaction for any schedule settings.
Also check if there's a dedicated daily-flush cron job already.
</task>

<task id="2" name="Add daily flush cron" depends="1">
Add a cron job to openclaw.json that triggers memory flush at 23:00 UTC.
This is in addition to the compaction-triggered flush (which fires based on token threshold).
The cron should use a session payload that triggers the flush prompt.
</task>

<task id="3" name="Verify cron configuration" depends="2">
Read back the cron config to confirm the daily flush is scheduled.
Check that the schedule is "0 23 * * *" (23:00 UTC daily).
</task>

## Verification
- [ ] Daily flush cron fires at 23:00 UTC
- [ ] Cron produces summaries reflecting the day's activity
- [ ] Existing compaction-triggered flush still works as backup

## must_haves
- Daily flush cron at 23:00 UTC producing day-end summaries
