4. Edit the config. Use python3 to safely insert the job (JSON5 parsing can be tricky):
```bash
# Read current jobs, add new one, write back
python3 << 'PYEOF'
import json, re, sys

# Read config (strip JSON5 comments for parsing)
with open('/home/ubuntu/.openclaw/openclaw.json', 'r') as f:
    content = f.read()

# Strip // comments and trailing commas for JSON parsing
clean = re.sub(r'//.*$', '', content, flags=re.MULTILINE)
clean = re.sub(r',\s*([}\]])', r'\1', clean)

config = json.loads(clean)

# Navigate to jobs array
if 'cron' in config and 'jobs' in config['cron']:
    jobs = config['cron']['jobs']
elif 'jobs' in config:
    jobs = config['jobs']
else:
    print("ERROR: Cannot find jobs array in config")
    sys.exit(1)

# Check if already exists
if any(j.get('id') == 'auto-brief-gen' or j.get('name') == 'auto-brief-gen' for j in jobs):
    print("auto-brief-gen already exists, skipping")
    sys.exit(0)

# Add new job
new_job = {
    "id": "auto-brief-gen",
    "name": "auto-brief-gen",
    "cron": "30 9 * * 2,5",
    "tz": "America/Los_Angeles",

jobs.append(new_job)

PYEOF
```

**Part B: Create host-level git push script**

Since the Docker sandbox may not have git or GitHub credentials, create a host-level script that pushes research-briefs to GitHub. This script runs outside the sandbox.

Create ~/clawd/scripts/push-research-briefs.sh:
```bash
#!/bin/bash
# Push new research briefs to pops-claw GitHub repo
# Called by crontab after auto-brief skill runs, or manually
set -euo pipefail

REPO_DIR="$HOME/clawd/pops-claw-repo"
LOG="$HOME/clawd/scripts/push-research-briefs.log"

cd "$REPO_DIR"

# Pull latest first
git pull origin main --ff-only 2>/dev/null || {
  echo "[$(date '+%Y-%m-%d %H:%M')] Pull failed, aborting push" >> "$LOG"
  exit 1
}

# Check for changes in research-briefs/
if git diff --quiet HEAD -- research-briefs/ && [ -z "$(git ls-files --others --exclude-standard research-briefs/)" ]; then
  echo "[$(date '+%Y-%m-%d %H:%M')] No new briefs to push" >> "$LOG"
  exit 0
fi

# Stage, commit, push
git add research-briefs/
BRIEF_COUNT=$(git diff --cached --name-only -- research-briefs/ | wc -l | tr -d ' ')
git commit -m "briefs: auto-generate ${BRIEF_COUNT} new research brief(s)"
git push origin main

echo "[$(date '+%Y-%m-%d %H:%M')] Pushed ${BRIEF_COUNT} brief change(s) to GitHub" >> "$LOG"
```
