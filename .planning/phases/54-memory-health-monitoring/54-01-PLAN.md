# Plan 54-01: Create Memory Health Check Script

## Objective
Create a shell script that verifies the memory system is working: daily logs exist, QMD indexed them, and search returns results.

## Tasks

<task id="1" name="Write health check script">
Create ~/scripts/memory-health-check.sh that:
1. Checks yesterday's daily log exists at ~/clawd/agents/main/memory/YYYY-MM-DD.md
2. Runs QMD search for a known term and verifies non-empty results
3. Checks MEMORY.md exists and is not empty
4. Logs pass/fail with timestamp to ~/scripts/memory-health-check.log
5. Exits 0 on all pass, 1 on any failure with specific error message
</task>

<task id="2" name="Deploy to EC2" depends="1">
SCP script to EC2, chmod +x, run once to verify.
</task>

<task id="3" name="Verify script passes" depends="2">
Run the script and confirm exit code 0.
Check log file for clean output.
</task>

## Verification
- [ ] Script verifies daily log existence
- [ ] Script verifies QMD search returns results
- [ ] Script verifies MEMORY.md exists
- [ ] Script logs results and exits with correct code

## must_haves
- Health check script that verifies daily logs, QMD indexing, and search
