**Step 2: Initialize Build**

  ```python

  from datetime import datetime
  conn = sqlite3.connect('/workspace/yolo-dev/yolo.db')
  conn.row_factory = sqlite3.Row
  ```

  ```python

      INSERT INTO builds (date, slug, name, description, status, tech_stack, started_at)
      VALUES (?, ?, ?, ?, 'idea', ?, ?)
  """, (datetime.now().strftime('%Y-%m-%d'), slug, name, description, tech_stack,
        datetime.now().isoformat()))
  conn.commit()
  conn.close()
  ```
- UPDATE status to 'building':
  ```python

  conn.execute("UPDATE builds SET status='building' WHERE id=?", (build_id,))
  ```

- Determine final status:
  - Score 4-5 → status = 'success'
  - Score 3 → status = 'partial'
  - Score 1-2 → status = 'failed'

  ```python
      UPDATE builds SET
          status=?, tech_stack=?, lines_of_code=?, files_created=?,
          self_score=?, self_evaluation=?, build_log=?, error_log=?,
          completed_at=?, duration_seconds=?

  """, (final_status, tech_stack, total_loc, file_count,
        score, evaluation_text, stdout_log, stderr_log,
        datetime.now().isoformat(), duration_secs, build_id))
  conn.commit()
  conn.close()
  ```
