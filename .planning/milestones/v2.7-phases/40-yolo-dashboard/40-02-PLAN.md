### B. Create `src/app/yolo/page.tsx`

`"use client"` directive at top.

**Data fetching:** `useSWR<YoloBuildSummary>("/api/yolo/builds", fetcher)` where fetcher is `(url: string) => fetch(url).then(r => r.json())`. SWR inherits global 30s refreshInterval from providers.tsx config.

**Filter state:** `useState<"all" | "success" | "partial" | "failed">("all")`. Filter builds: `filter === "all" ? builds : builds.filter(b => b.status === filter)`.

**Layout:**
```
<div className="min-h-screen px-6 py-10">
  <div className="max-w-7xl mx-auto space-y-8">
    {/* Header */}
    <div>
      <h1 className="text-2xl font-bold">YOLO Builds</h1>
      <p className="text-muted-foreground mt-1">Autonomous overnight prototypes</p>
    </div>

    {/* Filter bar */}
    <div className="flex items-center gap-2">
      {FILTERS.map(f => pill button with counts)}
    </div>

    {/* Grid or loading/empty state */}
    <div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
      {filtered.map(build => <YoloBuildCard key={build.id} build={build} />)}
    </div>
  </div>
</div>
```

**Filter bar pills:** Array of `{ key: "all"|"success"|"partial"|"failed", label: string }`. Active state: `bg-secondary text-foreground`. Inactive: `text-muted-foreground hover:bg-secondary/50 hover:text-foreground`. Each non-"all" pill shows count from `data.counts[key]` as a small `text-xs text-muted-foreground` span after the label. Use `cn()` for conditional classes. `rounded-full px-3 py-1 text-sm font-medium transition-colors`.

**Loading state:** When `!data && !error`, show 3 skeleton cards. Each skeleton: `Card` with `animate-pulse` containing gray placeholder divs (`h-4 w-3/4 bg-muted rounded`, `h-3 w-full bg-muted rounded`, `h-3 w-1/2 bg-muted rounded`).

**Empty state:** When `data.builds.length === 0` (or filtered is empty), centered `<p className="text-muted-foreground text-center py-12">No builds yet</p>`. If filtering and no results: "No {filter} builds".

**Error state:** Fallback gracefully — SWR error still shows empty grid with "No builds yet".

Import `YoloBuild`, `YoloBuildSummary` from `@/lib/queries/yolo` for type annotation. Import `useSWR` from `swr`.
