## Milestone Summary

**Key Decisions:**
- Python sqlite3 module for sandbox DB access (sandbox constraints)
- Docker nested bind-mounts unreliable — use workspace subdirectories
- yolo-dev canonical path: ~/clawd/agents/main/yolo-dev/ (served via main workspace mount)
- Isolated cron sessions need explicit bind mounts in openclaw.json
- Client-side filtering for <100 builds (not server-side query params)
- Health dots over Badge pills for compact table density
